# Today I Learned (TIL)

Today I Learned Started by. 2024 / 4 / 15

このリポジトリは、日々の学習や発見を記録するためのものです。
フロントエンド開発、3D グラフィックス、ウェブ技術など、さまざまな分野の知識やスキルをここに蓄積しています。

## 構造

この TIL は以下のカテゴリに分類されています：

- `dev-tools/`: 開発ツールに関する知識
  - `markdown/`: Markdown 記法
  - `vscode/`: Visual Studio Code の使用方法
- `frontend/`: フロントエンド開発
  - `3d-graphics/`: 3D グラフィックス（Three.js 等）
  - `frameworks/`: フレームワーク（Next.js 等）
  - `javascript/`: JavaScript 言語
  - `markup/`: HTML と CSS
- `web/`: ウェブ技術全般
  - `api/`: API 設計と使用
  - `http/`: HTTP プロトコル
  - `internet/`: インターネットの基礎
