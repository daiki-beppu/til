# 拡張機能: Markdown All in One の使い方

## Markdown All in One のすごいところ

- キーボードショートカットを使えるようになる
- 目次の自動生成
- 見出しのナンバリング
- 補完機能

### キーボードショートカットが使えるようになる

| ショートカットコマンド | 説明                 |
| ---------------------- | -------------------- |
| ⌘ + B                  | 太字 bold            |
| ⌘ + I                  | 斜体 italic          |
| ⌘ + shift + [          | 見出しレベルを上げる |
| ⌘ + shift + ]          | 見出しレベルを下げる |

### 目次の自動生成

見出しレベルで目次を自動生成してくれる

1. コマンドパレットを開く ⌘ + shift + P
1. コマンドパレットに `create table of contents`と入力(一部でも表示される)
1. Enter を 押す

### 見出しのナンバリング

見出しに自動で番号をふってくれる

1. コマンドパレットを開く ⌘ + shift + P
1. コマンドパレットに `section number`と入力(一部でも表示される)
   1. `add/updete seciton number` で追加
   1. `Remove seciton nunber`で削除
