# HTTP リクエスト

HTTP リクエストを投げるときに URL は大事な役割を持つ
1 文字でも間違えば適切な場所にリクエストは送れない

## クエリストリング

URL の末尾に ? から始まる文字列のこと
クエリストリングは URL にいろいろな情報を乗せることができる
URL と違って何を記述して大丈夫
エンドポイントとしては基本的にはスルーされる
バックエンドがほしい情報だけが使われるので問題ない

基本の形はキーと値のセットで & でつなげて記述されている

こちらは JavaScript の検索結果 URL の一部

https://www.google.com/search?
`q=JavaScript` キーが q 値が JavaScript
`&` アンパサンドでつなげている
sca_esv=345267d81bec8f30&
