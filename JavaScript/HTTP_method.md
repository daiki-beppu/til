# HTTP メソッドについて

HTTP メソッドとは
Web サーバーに対して行うリクエストの種類を指定する命令のこと

## HTTP メソッドの種類

- リソースの取得を要求 `GET`
  特定の情報、Web ページや画像などのリソースを取得する

- サーバーにデータを送信して処理を行う `POST`
  フォームの送信やデータの登録などの処理を行う

- 指定した URL のリソースを更新する `PUT`
  既存のデータをリクエストしたデータに置き換える

- 指定した URL のリソースを削除する `DELETE`

- 指定した URL のリソースの一部を更新する `PATCH`
  `PUT`との違いはリソース全体ではなく**一部**の更新
