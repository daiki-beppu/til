# web の概要(超ざっくり)

## web とは

web とは "World Wide Web"の略で
インターネットを介して文書などのリソースを提供するもの

リソースには URL が割り当てられている
URL は " Uniform Resource Locater "の略

インターネット上の web ページやファイルの位置や場所を示すもの

### HTTP について

HTTP とは "Hyper Text Transfer Protocol"
直訳するとハイパーテキストを転送するプロトコル

プロトコルとは PC 同士がコミュニケーションを取るための
標準化された手順や規格のこと
