# API 入門

API とは
Application Programming Interface の略
プログラム同士を繋ぐためのインターフェースのこと
ソフトウェアアプリケーションが他のアプリやサービスと通信してデータのやりとりをしたり、機能を利用するために用いられる

## web における API

web における API (webAPI) とは
HTTP リクエストを使って外部のサービスの通信しデータのやりとりをしたり、機能を利用するために用いられる

**特定の URL に対してリクエストを投げるこの URL をエンドポイント**と呼ぶ

様々な企業が webAPI のエンドポイントを公開している

このエンドポイントからは様々な情報を取得することができ
webAPI は別のアプリケーションの入口のようなものだと考えられる
