# Next.js 14 環境構築

> [!NOTE]
> パッケージマネージャーは bun を使用
> bun のインストールは `npm install -g bun`

## ターミナルで `bunx create-next-app@latest` を叩く

![](https://i.gyazo.com/30d3a9f67e1f2bd29797a4e9bc6dbf91.png)

- What is your project named? プロジェクト名を入力 今回は`demo-site`と入力
- Would you like to use TypeScript? TypeScript を使いますか？ `Yes`
- Would you like to use ESLint? ESlint を使いますか？ `Yes`
- Would you like to use Tailwind CSS? … Tailwind CSS を使いますか？ `Yes`
- Would you like to use `src/` directory? … src ディレクトリを使いますか？ `No`
- Would you like to use App Router? (recommended) … App Router を使いますか？ `Yes`
- Would you like to customize the default import alias (@/\*)? …
  インポートエイリアスをカスタマイズしますか？ `No`
